-jar : java 라이브러리 종류중 하나
-asset : 외부소스 추가할때 
- res : 자원(화면을 구성하는 자원)
- layout : 정적 프로그래밍
- Manifest : 설명서 같은 느낌, 퍼미션 : 허락 / 

- 4가지 액티비티 : 한 화면 / 서비스: 백그라운드 / 브로드캐스트리시버 : 알림, / 
	컨텐트프로바이더 : 그림 , 동영상 제공해주는 부분

- Intent filter : 실행시켜줄때 사용한다.

- 버튼 5개 마다 다른 토스트 메시지 실행하기 : 숙제

- 인터페이스 : 사용자와 컴퓨터간의 상호작용...

- 온클릭 리스너 구현한 것이 인터페이스라고 할 수 있다.
